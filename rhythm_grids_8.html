Rhythm Grids: 8th Note Subdivisions


♪ Step 1: You will receive a random two measure rhythm in 4/4. The smallest value is an eighth note. The grid below is divided into eighth note value subdivisions, and color-coded by measure for clarity.

♪ Step 2: If a rhythm enters on a particular beat (grid box), select it with the corresponding answer menu.
If a rhythm lasts for more than one box (half a beat), add the rhythm under the first box it enters on, then fill the remaining boxes with arrow symbols (→). This shows the total length of the rhythm.

♪ Step 3: When you're done, press "Submit." The correct grid will be revealed for you. Practice counting the grid numbers out loud and clapping when you see a note in a corresponding box.

♪ Step 4: Keep practicing!
